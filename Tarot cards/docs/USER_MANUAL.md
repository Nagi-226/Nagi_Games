# 塔罗牌占卜系统 - 用户操作手册

欢迎使用 **✦ 塔罗牌占卜系统 ✦**！这是一个提供78张标准塔罗牌的数字化在线占卜平台。

---

## 快速入门指南

### 1. 开始一次新的占卜
1. 打开应用，进入**首页**。
2. 在“选择牌阵”区域，您可以看到系统提供的多种经典牌阵：
   - **单牌占卜** (1张牌)：获取每日指引。
   - **三卡牌阵** (3张牌)：解读过去、现在和未来。
   - **五牌阵** (5张牌)：深入分析现状和结果。
   - **关系牌阵** (5张牌)：专门用于感情和人际关系。
   - **凯尔特十字** (10张牌)：最全面、最复杂的分析。
3. 点击您喜欢的牌阵卡片，即可进入**占卜界面**。

### 2. 抽牌与解读
1. 进入占卜页面后，您可以（可选地）在输入框内输入您想询问的问题。
2. 系统将为您洗好一副完整的78张塔罗牌，并呈现在上方牌堆中。
3. **点击**牌堆中发光的塔罗牌进行抽取，直到抽满该牌阵所需的牌数。
4. 抽取完成后，塔罗牌会放置在对应的牌阵位置上（牌面朝下）。
5. 此时，**点击**下方牌阵中的任意一张牌，即可触发翻牌动画，并展示该位置的详细牌意解读。
6. 牌意包括：正/逆位、关键词、正位含义、逆位含义、象征符号、元素和占星对应。

### 3. 查看塔罗牌库
如果您想单独学习和了解每张塔罗牌的含义，可以访问顶部的 **牌库** 导航。
1. 在这里，您可以浏览全部78张牌。
2. 使用顶部的**筛选按钮**，您可以按类型（大阿卡那、小阿卡那）或按花色（权杖、圣杯、宝剑、星币）进行过滤。
3. 点击任意一张牌，会弹出一个窗口显示详细的教学信息。

### 4. 历史记录管理
您的每次占卜结果都会自动保存到系统中。
1. 点击顶部导航栏的 **历史记录**。
2. 页面会按时间顺序列出您所有过去的占卜。
3. 点击任意一条记录，可以回顾当时的问题、抽到的牌以及牌阵位置。
4. 如果需要，您可以点击记录下方的 **删除** 按钮移除不需要的历史。

## 常见问题 (FAQ)

**Q: 塔罗牌为什么会有正位和逆位？**
A: 逆位通常表示能量受阻、内在化或含义的反转。本系统在洗牌时会有30%的概率出现逆位，这为占卜增加了更多的解读维度。

**Q: 我可以一次抽取所有牌吗？**
A: 系统为了模拟真实的占卜体验，需要您亲手从牌堆中逐一挑选卡牌。

**Q: 数据会丢失吗？**
A: 所有历史记录都会保存在后端的 SQLite 数据库中，只要服务器不被重置，您的占卜记录就会永久保留。
