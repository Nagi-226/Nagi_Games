# 塔罗牌占卜系统 - API文档

## 基础 URL
所有API请求的基础URL为: `/api`

---

## 1. 塔罗牌资源 (Cards)

### 1.1 获取所有塔罗牌
- **URL**: `/cards`
- **方法**: `GET`
- **描述**: 返回所有78张塔罗牌的详细信息。
- **查询参数**:
  - `type` (可选): 按类型过滤 (`major` 或 `minor`)
  - `suit` (可选): 按花色过滤 (`wands`, `cups`, `swords`, `pentacles`)

### 1.2 获取单张塔罗牌
- **URL**: `/cards/:id`
- **方法**: `GET`
- **描述**: 根据ID返回特定塔罗牌的详细信息。
- **参数**:
  - `id`: 塔罗牌的唯一标识符 (0-77)

### 1.3 搜索塔罗牌
- **URL**: `/cards/search/:query`
- **方法**: `GET`
- **描述**: 根据关键词、名称或牌意搜索相关塔罗牌。

---

## 2. 牌阵资源 (Spreads)

### 2.1 获取所有牌阵
- **URL**: `/spreads`
- **方法**: `GET`
- **描述**: 返回系统支持的所有牌阵布局。

### 2.2 获取特定牌阵
- **URL**: `/spreads/:id`
- **方法**: `GET`
- **描述**: 返回特定牌阵的详细信息（包括各个位置的含义）。

---

## 3. 占卜记录资源 (Readings)

### 3.1 创建新占卜记录
- **URL**: `/readings`
- **方法**: `POST`
- **描述**: 保存一次新的占卜结果。
- **请求体 (JSON)**:
  ```json
  {
    "spread_id": 1,
    "question": "我的未来如何？",
    "cards_drawn": [
      { "card_id": 0, "position": 1, "is_reversed": false },
      { "card_id": 15, "position": 2, "is_reversed": true }
    ]
  }
  ```

### 3.2 获取所有占卜记录
- **URL**: `/readings`
- **方法**: `GET`
- **描述**: 返回所有历史占卜记录，按时间倒序排列。

### 3.3 获取单条占卜记录
- **URL**: `/readings/:id`
- **方法**: `GET`
- **描述**: 获取某次占卜的详细结果。

### 3.4 删除占卜记录
- **URL**: `/readings/:id`
- **方法**: `DELETE`
- **描述**: 删除指定的占卜历史记录。
